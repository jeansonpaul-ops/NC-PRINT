⚙️ Configuration EmailJS

EmailJS permet d'envoyer des emails directement depuis le navigateur, gratuitement (200 mails/mois).
Une seule fois à configurer.

Comment obtenir ces clés (gratuit) :
1. Créer un compte sur emailjs.com
2. Ajouter un service email (Gmail, Outlook…) → copier le Service ID
3. Créer un template avec les variables {{rapport_html}} et {{date}} → copier le Template ID
4. Dans Account → API Keys → copier la Public Key
